# Czech Data Steward Community

Toto je oficiální organizační repozitář pro českou komunitu Data Stewardů.

## O komunitě

Česká komunita Data Stewardů sdružuje odborníky z oblasti správy dat, datového managementu a implementace FAIR principů ve výzkumu a praxi.

## About the Community

The Czech Data Steward Community brings together professionals in data management, data governance, and implementation of FAIR principles in research and practice.
